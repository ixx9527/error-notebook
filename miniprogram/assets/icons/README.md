# Tab Bar 图标说明

需要为 tab bar 提供以下 6 个 PNG 图标文件（建议 81×81px）：

| 文件名 | 说明 |
|--------|------|
| `home.png` | 首页 - 未选中（灰色） |
| `home-active.png` | 首页 - 选中（蓝色 #4A90D9） |
| `book.png` | 错题 - 未选中（灰色） |
| `book-active.png` | 错题 - 选中（蓝色 #4A90D9） |
| `user.png` | 我的 - 未选中（灰色） |
| `user-active.png` | 我的 - 选中（蓝色 #4A90D9） |

可从 [iconfont.cn](https://www.iconfont.cn/) 下载，或使用 iconfont 字体方案替代。
